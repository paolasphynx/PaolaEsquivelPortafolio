# 🎨 Paola Esquivel | Portfolio Creativo

Este es el repositorio oficial del portfolio profesional de **Paola Esquivel**, un espacio diseñado para exhibir proyectos de diseño visual, branding y estrategia digital. El sitio está construido con un enfoque en la experiencia de usuario (UX), interactividad fluida y optimización SEO.

---

## 🚀 Tecnologías Utilizadas

El proyecto combina herramientas modernas para lograr un rendimiento óptimo y un diseño responsivo:

* **Frontend:** HTML5, CSS3 (Arquitectura modular).
* **Frameworks:** [Bootstrap 5](https://getbootstrap.com/) para la estructura grid.
* **Interactividad:** [Swiper.js](https://swiperjs.com/) para sliders táctiles y [Lucide Icons](https://lucide.dev/) para iconografía minimalista.
* **Tipografía:** Plus Jakarta Sans vía Google Fonts.

---

## 🛠️ Estructura del Proyecto

Para mantener el código limpio y escalable, los estilos se han dividido por secciones:

```text

├── css/
│   ├── hero.css           # Sección de bienvenida
│   ├── nav.css            # Navegación pegajosa y móvil
